About

Hello! Welcome to zsdanalytics.com! This site is my personal project for practicing programming. The site itself is run using a Flask back-end connected to a front-end that utilizes JavaScript React for the interactive UI and designed using Bootstrap. The baseball portion of the site connects to a MySQL database that uses Python code to calculate statistics and analyze the data with the help of popular libraries such as Pandas, NumPy, and more.

Baseball

Looking for more in-depth statistics than regular box scores? The baseball section of this site is aimed at providing zone-specific and situational analysis of players. There is currently support for data from the 2019 regular season and statistics for pitchers. More features are planned - including batter support, player comparison, pitcher-batter matchups, and additional season support (with postseason data)!
